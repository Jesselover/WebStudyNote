# #  key Takeaways

- 视角控制 ：鼠标中键
- 视图切换 ：
	数字  / ctrl + 数字 ; 
	9 反转当前视图；
	0 切换摄像机视角；
	5 切换透视投影和正交投影；
	 2，4，5，6 ：前后左右微调；
	  +  -  缩放微调;
- 物体控制：
	移动物体：G ；旋转物体：R；缩放物体：S；恢复变换：ALT + G / R / S；
